Frequently asked questions

  • Is Sakura safe?
    we are open source so you can modify the code and compile yourself, we are trusted by the Community.
  • What does Sakura mean?
    Sakura means "Cherry blossom" that is Japanese.
  • What websites do we own?
    currently we only own notreasy.github.io, any other sites may have malware or be dangerous.
  • Will Sakura work on my computer?
    Sakura only supports 64bit windows.
  • Is Sakura liquidbounce?
    Sakura is a fork of liquidbounce so you get the benefits of liquidbounce while still getting updates and support.
  • Where is the Sakura source code?
    Sakura's source code can be found on github.
